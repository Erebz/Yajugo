Documentation des données membres

◆ caractere

string Case::caractere

Chaîne de caractères à afficher pour représenter la case

Note
On ne pouvait pas utiliser un char à la place d'un string car il ne permet pas d'afficher les caractères spéciaux qui étaient nécessaire pour l'affichage.

◆ id

unsigned int Case::id = 0

Identifiant d'une case (= sa catégorie: un mur, un chemin, un bloc infranchissable...) (0 par défaut, la case est défini comme étant un mur)

◆ objet

Objet* Case::objet = NULL

Pointeur vers un Objet (à NULL par défaut, la case ne contient pas d'objet à sa création)

◆ traversable

bool Case::traversable = false

Booléen indiquant si la case est traversable par une entité (joueur ou ennemi) (false par défaut, la case est infranchissable à sa création)
