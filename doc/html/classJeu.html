◆ iemeEnnemiUtiliseCompetence()

bool Jeu::iemeEnnemiUtiliseCompetence ( unsigned int  i,
Competence *  comp,
unsigned int &  degats 
)

Applique une compétence connue d'un ennemi$.

Si la compétence passe le test de précision, la fonction renvoie true et la compétence est effectivement appliquée. Les paramètres en données-résultats sont modifiés en conséquence.
Si la compétence échoue au test de précision, la fontion renvoie false, la compétence n'est pas appliquée et les paramètres en données-résultats ne sont pas modifiés.
La compétence et l'indice de l'ennemi sont passés en paramètres.

Paramètres
[in,out]iun unsigned int, indice de l'ennemi qui utilise la compétence
[in,out]compun pointeur de Compétence, la compétence utilisée
[in,out]degatsun unsigned int, le nombre de dégâts que produit la compétence
Renvoie
Un booléen, true si la compétence passe le test de précision, false si elle échoue.

◆ joueurUtiliseCompetence()

bool Jeu::joueurUtiliseCompetence ( Competence *  comp,
unsigned int &  nbEnnemisTouches,
string *&  ennemis,
unsigned int *&  degats,
unsigned int &  xp 
)

Fonction: Traite l'utilisation d'une compétence.

Si la compétence passe le test de précision, la fonction renvoie true et la compétence est effectivement appliquée. Les paramètres en données-résultats sont modifiés en conséquence.
Si la compétence échoue au test de précision, la fontion renvoie false, la compétence n'est pas appliquée et les paramètres en données-résultats ne sont pas modifiés.
La compétence utilisée est passée en paramètre.

Paramètres
[in,out]compun pointeur vers une Competence, pointe vers la compétence que le joueur veut utiliser
[in,out]nbEnnemisTouchesun unsigned int, compte le nombre d'ennemis touchés (si la compétence passe le test de précision)
[in,out]ennemisun pointeur vers un tableau de string, chaîne de caractères, listant les noms des ennemis touchés par la compétence respectivement (si la compétence passe le test de précision)
[in,out]degatsun pointeur vers un tableau d'unsigned int, listant les dégâts des ennemis touchés par la compétence respectivement (si la compétence passe le test de précision)
[in,out]xpun unsigned int, somme les points d'expérience gagnés des ennemis vaincus par la compétence (si la compétence passe le test de précision)
Renvoie
Un booléen, true si la compétence passe le test de précision, false si elle échoue.
